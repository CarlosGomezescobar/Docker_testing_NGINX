Bienvenido a Docker

Maquinas Virtuales

Son software que emulan o simulan el funcionamiento de un ordenador real con sus componentes de hardware, sistema operativo y aplicaciones.

Docker

Docker es una plataforma de codigo abierto que permite crear, usar y ejecutar aplicaciones dentro de contenedores de software.

Beneficios de Docker

Aislamiento, Portabilidad, Reutilizacion, Colaboracion.
